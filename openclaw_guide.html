一、Gateway 管理（最常用）

🔴 Gateway 服务控制

命令功能常用场景
openclaw gateway status查看 Gateway 服务状态 + 连通性检测每次连接前必查
openclaw gateway start启动 Gateway 服务服务未运行时
openclaw gateway stop停止 Gateway 服务重启前或关闭时
openclaw gateway restart重启 Gateway 服务配置变更后
openclaw gateway run前台运行 Gateway（看日志）调试/排查问题时
openclaw gateway health查询 Gateway 健康状态检查服务是否正常

🌐 Gateway 网络配置

命令功能说明
openclaw gateway start --bind lan绑定 LAN IP（局域网访问）其他设备连接
openclaw gateway start --bind loopback仅本地回环（默认）安全模式
openclaw gateway start --port 18789指定端口端口冲突时
openclaw gateway start --token XXXX设置访问 Token认证模式
openclaw dashboard打开 Web 控制台浏览器管理界面
Gateway bind 模式说明：
• loopback：仅本机访问（默认，安全）
• lan：绑定 0.0.0.0，局域网可访问
• tailnet：通过 Tailscale 网络访问（需配置 tailscale mode）
• auto：自动选择最佳模式

💡 典型 Gateway 启动流程

# 1. 查看当前状态 openclaw gateway status # 2. 启动服务（局域网模式） openclaw gateway start --bind lan # 3. 打开控制台 openclaw dashboard # 4. 查看实时日志 openclaw logs

二、Skills 管理

命令功能说明
openclaw skills list列出所有已安装 Skills查看当前环境
openclaw skills search <关键词>搜索 ClawHub 上的 Skills找新工具
openclaw skills install <url>从 ClawHub/GitHub 安装 Skill安装新 Skill
openclaw skills info <skill名>查看 Skill 详细信息了解 Skill 用途
openclaw skills check检查 Skills 依赖是否满足排查问题用
openclaw skills update更新所有已安装的 Skills保持最新

💡 Skills 安装示例

# 从 ClawHub 安装（完整包名） openclaw skills install clawhub.ai/username/skill-name # 从 GitHub 安装 openclaw skills install https://github.com/user/repo # 查看已安装的 Skills openclaw skills list

五、Channels 渠道管理

命令功能说明
openclaw channels list列出所有已配置渠道查看连接状态
openclaw channels status查看渠道详细状态排查连接问题
openclaw channels add <参数>添加渠道账号配置新渠道
openclaw channels login <渠道>登录渠道账号授权连接
openclaw channels logout <渠道>登出渠道账号断开连接
openclaw channels logs查看渠道日志调试问题
openclaw channels resolve <名称>解析渠道/用户 ID获取对话 ID

💡 添加渠道示例

# 添加 Telegram Bot openclaw channels add --channel telegram --token <你的Bot Token> # 添加 QQ 频道 Bot openclaw channels add --channel qqgroup --app-id <AppID> --token <Token> # 查看渠道状态 openclaw channels status

六、Cron 定时任务

命令功能说明
openclaw cron list列出所有 Cron 任务查看已配置的定时任务
openclaw cron add <参数>添加新 Cron 任务创建定时任务
openclaw cron rm <id>删除 Cron 任务移除定时任务
openclaw cron run <id>立即运行 Cron 任务调试/手动触发
openclaw cron runs <id>查看任务执行历史检查运行结果
openclaw cron status查看 Cron 调度器状态检查是否正常
openclaw cron disable <id>禁用 Cron 任务暂停定时任务
openclaw cron enable <id>启用 Cron 任务恢复定时任务

💡 Cron 任务添加示例

# 添加每天早上9点执行的定时任务 openclaw cron add --name "daily-report" --schedule "0 9 * * *" --message "生成今日竞品报告" --agent pet-macos # 列出所有 Cron 任务 openclaw cron list # 手动触发一次 openclaw cron run <任务ID>
