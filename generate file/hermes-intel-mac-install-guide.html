Hermes Agent Intel Mac 安装指南

pet-competitor profile 专用

1. 安装 Python 3

macOS 系统自带 Python，但建议使用独立安装的 Python 3。

# 使用 Homebrew 安装（推荐） brew install python3 # 验证安装 python3 --version

2. 安装 Hermes Agent

macOS 不需要虚拟环境，直接安装即可。

# 安装 Hermes pip3 install hermes-agent # 验证安装 hermes --version

3. 配置 Profile

pet-competitor profile 已配置完毕，配置文件位于：

~/.hermes/profiles/pet-competitor/config.yaml

主要配置包括：

- providers: 配置了 MiniMax 作为默认 provider - platforms: 配置了 WeChat 平台 - custom_providers: minimax-cn（base_url: https://api.minimaxi.com/v1） - memory: 存储宠物行业研报、全球动保五强财报等知识库

4. 配置 QQBot

在 config.yaml 中配置 QQBot 平台：

platforms: qqbot: enabled: true bot_token: "YOUR_APPID:YOUR_APPSECRET"

配置步骤：

1. 前往 QQ开放平台 (https://q.qq.com) 创建应用
2. 获取 AppID 和 AppSecret
3. 在 config.yaml 中填写：bot_token: "YOUR_APPID:YOUR_APPSECRET"
4. 重启 Hermes Agent 使配置生效

5. 配置 WeChat Bot

在 config.yaml 中配置 WeChat 平台：

platforms: weixin: enabled: true bot_token: "YOUR_TOKEN:YOUR_AESKEY"

配置步骤：

1. 前往微信公众平台 (https://mp.weixin.qq.com) 配置服务器信息
2. 获取 Token 和 EncodingAESKey
3. 在 config.yaml 中填写：bot_token: "TOKEN:AESKEY"
4. 重启 Hermes Agent 使配置生效

6. 启动 Hermes

# 使用 pet-competitor profile 启动 hermes profile use pet-competitor # 直接启动 hermes run --profile pet-competitor &

7. 验证运行状态

# 检查进程 ps aux | grep hermes # 查看日志 tail -f ~/.hermes/logs/gateway.log

8. 更新 Hermes

# 升级到最新版本 pip3 install --upgrade hermes-agent # 重启服务使新版本生效
