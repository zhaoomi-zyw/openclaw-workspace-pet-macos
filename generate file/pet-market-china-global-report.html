执行摘要

面向管理层的结论先行版：如果只看对营销最重要的信号，当前最值得押注的不是“总量爆发”，而是“猫化、预防化、全年化、组合化”。

1. 中国总量仍在涨，但增量更偏猫

2020-2024 年中国城镇犬猫总量从 10084 万只增至 12411 万只；其中猫由 4862 万只增至 7153 万只，狗基本在 5100-5400 万只平台震荡。新增宠物更偏向猫，意味着更适合室内、年轻人、精细护理的产品组合持续受益。

2. 驱虫已是宠物健康消费的高频入口

公开行业摘要显示，2024 年中国宠物驱虫药市场规模约 336.09 亿元，且驱虫已经是宠物健康医疗中的第一大核心品类，占比 20.7%。对营销而言，驱虫不只是药品，更是复购型健康管理入口。

3. 海外成熟市场给出的不是“更多宠物”，而是“更强预防”

美国和欧洲的成熟度更高，公开材料里最明显的趋势不是数量暴涨，而是宠物被更深度纳入家庭预算，预防型营养、驱虫、补充剂和全年管理方案更容易被接受，这对中国未来 3-5 年有较强参考意义。

4. 2026-2030 中国增长会放缓，但高价值化会继续

本报告透明测算显示，到 2030 年中国城镇犬猫总量有望到 1.37 亿只左右，其中猫接近 8200 万只。数量并非高速爆发，但“单宠预防护理支出”仍有继续上行空间，因此营销重点应从单次拉新转向持续教育与复购留存。

2026-2030 趋势测算

测算逻辑非常保守：不照搬任何单一商业报告，而是基于 2020-2024 中国实际走势、猫狗结构变化、成熟市场对比以及消费升级信号做透明推演。 结果更适合做营销资源配置，而不是替代财务预算模型。

狗：低速平台期

狗在中国仍是重要市场，但 2020-2024 更像平台波动而非高速扩容，因此 2026-2030 采用 0.7%-1.1% 的低速增长假设。

猫：持续跑赢大盘

猫具备更适合一二线城市、年轻人、小空间、低时间投入的特征，因此 2026-2030 采用 2.2%-3.1% 的放缓式增长假设。

驱虫：增速快于宠物数量

驱虫消费不只受数量影响，还受全年预防心智、猫室内防护教育、组合型产品和单宠年均支出上行影响，因此预期其市场增速高于宠物数量增速。

营销解读

如果你是驱虫药、功能营养、医药电商或宠物医院品牌，最重要的不是“量”，而是“每只宠物被更长期管理”。

猫增长意味着什么

猫占比走高会天然放大“室内寄生虫风险教育”“滴剂/口服便利性”“应激更低的给药体验”和“线上内容教育”的重要性。

狗市场意味着什么

狗量增长趋缓并不等于机会变小。户外活动频率更高、蜱虫和心丝虫教育更成熟、组合型产品接受度更高，决定了狗仍是高客单、高复购的核心盘。

真正值得投入的环节

全年防护认知、季度/年度购包、医院和电商联动、内容种草到处方转化的闭环，会比单纯跑新增更有效。

猫狗结构变化

结构变化决定营销方式。中国最明显的是“猫化”，日本是“总量趋稳但猫长期高于狗”，美国则体现为猫家庭和狗家庭同时扩张，说明宠物不再只是单一陪伴对象，而是进入家庭长期消费系统。

2020 年猫占比仍低于 50%，到 2024 年已升至 57.6%，2030 测算接近 60%。

结构变化对驱虫营销的影响

重点不是简单复制犬用策略，而是把猫主人的决策链重新设计一遍。

内容重点

室内猫也需要预防、幼猫首驱、成年猫全年管理、内外同防的便利性，会比硬卖 SKU 更有效。

渠道重点

狗更适合医院、线下连锁与户外场景联动；猫更适合内容平台、直播电商和“低干扰、高复购”运营。

产品重点

从一次性驱虫向套餐化、订阅化、日历提醒和年度计划迁移，才更符合成熟市场经验。

驱虫相关消费 / 市场数据

驱虫需求与宠物数量增长相关，但它真正的放大器是“预防意识”和“全年管理习惯”。 因此，本节同时放入中国的市场规模线索和海外官方公开的依从性、教育缺口、企业销售信号。

对营销团队的直接建议

如果目标是做中国猫狗驱虫赛道的增量，建议优先从“教育 + 复购”设计，而不是单纯靠促销压货。

先讲风险，再讲产品

针对室内猫、幼宠、外出犬、家庭多宠，分别设计寄生虫风险内容。让用户先相信“需要全年防”，再谈品牌选择。

把驱虫从单品做成计划

季度装、半年装、年度计划、医院回访、会员提醒，是把一次成交变成长周期留存的关键动作。

猫和狗要拆开运营

狗的驱虫更适合“户外风险 + 医院处方”；猫更适合“室内也需预防 + 便捷低刺激给药 + 电商教育”。

把成熟市场经验本地化

海外启示不是直接照搬产品，而是学习“全年预防心智、长效方案、医生背书、补充剂与主粮协同”的运营方式。
