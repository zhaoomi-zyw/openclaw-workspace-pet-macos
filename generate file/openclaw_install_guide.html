OpenClaw 完整安装配置指南
环境准备 → 安装 → AI配置 → 渠道接入 → Skills → 记忆系统 · 全流程详解
v2026.3 · 适配 macOS/Linux
1
环境准备
# Node.js 要求 ≥ 18 node --version # → v24.14.1（当前版本） npm --version # → 11.11.0
✅ Node.js 已预装（macOS）
✅ npm 随 Node.js 一起安装
✅ Homebrew 可用（macOS）
未安装Node.js？
macOS: brew install node
Linux: curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
2
安装 OpenClaw
# 全局安装 OpenClaw CLI npm i -g openclaw # 或使用镜像（中国区） npm i -g openclaw --registry=https://registry.npmmirror.com
# 验证安装 openclaw --version # → 2026.3.31
✅ npm global bin 目录已自动加入 PATH
✅ openclaw 命令全局可用
3
配置 MiniMax M2.7
# 创建环境配置文件 touch ~/.openclaw/.env
# 写入 MiniMax API 配置 MINIMAX_API_HOST=https://api.minimaxi.com MINIMAX_API_KEY=sk-your-key-here TAVILY_API_KEY=tvly-your-key
# openclaw.json 添加模型 openclaw models add minimax/MiniMax-M2.7
✅ 当前已配置 api.minimaxi.com
✅ MiniMax-M2.7（2048k上下文，推理模式已开启）
4
接入 QQBot 频道
# 安装 QQBot 插件 openclaw plugin install qqbot
# 填写 QQ 开放平台 AppID 和 Token openclaw qqbot configure # 或手动编辑 ~/.openclaw/qqbot/config.json
# 当前配置（appId: 1903770990） # 账号: qqbot-pet-macos # 允许来源: *（所有用户）
✅ 插件目录: ~/.openclaw/qqbot/
✅ 会话数据: ~/.openclaw/qqbot/data/
首次配置需要扫码/填入Token，具体参考 QQ开放平台 开发者文档
5
安装 Skills
# 查看可用 Skills openclaw skills list
# 安装核心 Skills（clawhub） npx clawhub install web-access npx clawhub install minimax-mcp-call npx clawhub install minimax-multimodal npx clawhub install memory-setup npx clawhub install openclaw-tavily-search
✅ 当前已安装 6 个 skills
✅ 存储路径: ~/.openclaw/skills/
✅ skills-for-share/ 已同步至 GitHub
6
Workspace 与记忆系统
# 初始化 Workspace 目录 mkdir -p ~/.openclaw/workspace-pet-macos cd ~/.openclaw/workspace-pet-macos
# 创建核心配置文件 SOUL.md # AI 人格定义 AGENTS.md # 工作准则与流程 MEMORY.md # 长期记忆 memory/ # 每日日志目录
# 关联 GitHub 仓库 git remote add origin https://github.com/zhaoomi-zyw/openclaw-workspace-pet-macos.git git push -u origin main
✅ 仓库 Private，每次 commit 后自动 push
7
启动与运行
# 首次引导（onboard） openclaw onboard
# 启动 Gateway 守护进程 openclaw gateway start openclaw gateway status
# 查看所有命令 openclaw help
# 停止 Gateway openclaw gateway stop
推荐使用 screen/tmux 保持后台运行，或配置 systemd 服务
8
当前目录结构
~/.openclaw/ ├── .env ← API密钥（勿泄露） ├── openclaw.json ← 主配置 ├── qqbot/ ← QQ插件 ├── skills/ ← Skills目录 ├── memory/ ← 记忆存储 └── workspace-pet-macos/ ← 工作空间 ├── SOUL.md ├── AGENTS.md ├── MEMORY.md ├── memory/ └── skills-for-share/
✅ 配置完整性检查：openclaw status
✅ 敏感信息在 .env，不在 Git 追踪范围
重要：.env 已加入 .gitignore，切勿将 API Key 提交到 GitHub
