一、DTC战略解析 — 什么是"直达消费者"

DTC（Direct-to-Consumer）直达消费者

品牌绕过中间渠道（经销商、医院、零售商），直接触达终端消费者进行销售、运营和服务，建立用户数据资产，实现定价权和复购保障。

传统渠道 vs DTC渠道

药企 → 经销商 → 宠物店 → 宠主微信私域 → 宠主
药企 → 电商平台 → 宠主（被动）品牌旗舰店（主动运营用户）

DTC核心能力矩阵

  • 用户数据自持：拥有购买记录、用药习惯数据
  • 复购提醒：基于用药周期推送提醒
  • 订阅机制：定期自动发货（如AutoShip）
  • 私域运营：公众号/社群/小程序直接触达
  • 定价自主：不因渠道价格战损失溢价
  • 品牌忠诚：减少对医院推荐的依赖

DTC在中国宠物药市场的特殊意义

中国渠道特殊性：

  • 驱虫药以OTC为主，线上销售不受处方限制
  • 电商平台（天猫+京东）占据约25-30%市场份额
  • 年轻宠主（Z世代）60%+通过线上购买
  • 私域运营（微信）是高效复购工具

DTC的核心价值：

  • 驱虫药为慢病/预防属性，天然适合订阅
  • 月均复购，订阅可大幅提升LTV（生命周期价值）
  • 用户数据可指导产品研发和市场决策
  • 摆脱医院单一渠道，避免被平台/医院两端挤压

二、勃林格殷格翰 — DTC战略全景

核心定位：「医院+零售」双轨并行

勃林格的DTC策略与硕腾不同——不激进地绕开医院，而是"两条腿走路"：医院负责处方信任背书，零售/电商负责便捷复购，两者相互导流而非相互取代。

勃林格DTC在中国市场的核心动作

  • 2019年（全球首家外资）：在宠物驱虫领域，第一个开设天猫旗舰店的外资动保企业
  • 全系列上架：福来恩+犬心保+超可信+尼可信+博来恩，全部在旗舰店销售
  • 正品保障+科学养宠教育：旗舰店定位为"专业可信的购买渠道"而非纯货架
  • 2025年福味恩上市：与京东健康深度合作（润合供应链负责人出席）
  • 线上+医院双引流：电商购买链接可导流至医院核验处方

天猫旗舰店 — 运营定位

  • 官方授权：勃林格官方授权直营，确保100%正品
  • 全产品线展示：不仅是销售，更承担品牌教育和用户信任建立
  • 用户教育：宠物寄生虫危害科普，强化"预防>治疗"理念
  • 复购引导：用药周期提醒，促进年度复购
  • 与医院关系：不取代医院，而是"医院诊疗→旗舰店复购"的衔接

三、勃林格中国产品线 — 全部为OTC非处方药

重要事实：勃林格所有宠物驱虫药均为OTC（非处方药）

这一定位使其在天猫/京东等电商平台销售不受处方限制，是DTC战略的法律基础。如为处方药（RX），电商直接销售将面临合规风险。

博来恩 Broadline — 猫用独家

中国猫市场快速扩容（7153万只，2024），博来恩是唯一同时驱绦虫的猫用广谱驱虫药。

为什么OTC属性是DTC战略的关键？

可以做的事情：

  • 在天猫/京东直接开设旗舰店销售
  • 宠主无需处方即可线上购买
  • 订阅制/定期复购不受处方限制
  • KOL直播带货可合规进行

相比RX处方药的优势：

  • 不受处方流转系统监管限制
  • 可以进入任意电商/直播渠道
  • 复购路径无断点，用户体验完整
  • 内容营销（种草）不受合规约束

四、勃林格DTC渠道策略 — 三大支柱

支柱① 天猫旗舰店（2019年全球首家外资）

  • 定位：官方授权直营，确保正品，科学养宠教育阵地
  • 产品：全系列驱虫产品（福来恩/超可信/尼可信/犬心保/博来恩）
  • 差异化：vs硕腾（只卖大宠爱），勃林格一站式购齐
  • 用户价值：正品保障+专业推荐+便捷复购
  • 品牌背书：世界品牌论坛"年度品牌"三连冠
  • 与医院关系：不取代医院，而是"先医院后线上"的协同

支柱② 京东战略合作（2025福味恩上市）

  • 合作层级：高层战略合作（非单纯供货），润合供应链+京东健康共同出席
  • 产品首发：福味恩（FrontPro）作为2025年战略新品在京东首发
  • 渠道赋能：京东物流（正品+时效）+ 大数据用户洞察
  • 战略意图：借京东平台信任体系，强化"进口品质"定位，对抗国产品牌价格战
  • 内容营销：发布会上与"上海领养日"公益联动，强化品牌温度
  • 未来展望：京东+天猫双平台布局，形成互补

支柱③ 医院渠道双轨协同

  • 医院的核心价值：处方权威背书，特别是新用户首次购买时
  • 勃林格的策略：不激进地绕开医院，而是维护好医院渠道的同时，让医院成为DTC的"引流入口"
  • 超可信12周一次：大幅减少到诊频次，医院接受度高（减少推荐负担）
  • 合规优势：OTC产品不强制处方，医院可以推荐线上购买，患者自行选择
  • 挑战：医院存在"为他人做嫁衣"顾虑，需要利益共享机制

勃林格的独特优势

  • 超可信12周一次差异化：全球独家，减少给药频次本身就是DTC能力——"越少到诊，越依赖线上续购"
  • 全产品线天猫旗舰：vs硕腾单产品，勃林格给用户"一站式续购"的理由
  • 福来恩品牌溢价：三度世界品牌年度大奖，品牌信任支撑溢价，不陷入纯价格战
  • 医院+零售双轨：在维护医院关系的同时解放线上销售，渠道策略最务实
  • 全球资源背书：研发投入10%营收（2022年4.64亿欧元），产品迭代持续

勃林格的DTC短板

  • 私域运营薄弱：没有建立类似"品牌会员小程序"的私域资产，用户数据在平台手中
  • 订阅机制缺失：没有AutoShip功能，无法锁定年度复购
  • KOL/内容营销不足：勃林格的内容营销主要围绕发布会的PR稿，缺乏系统性
  • 下沉市场渗透：主要覆盖一二线城市，三四线城市线下渠道弱
  • 对平台依赖：天猫旗舰店的流量规则受平台算法控制，缺乏自主性

六、核心结论与战略建议

一句话结论

勃林格殷格翰是中国市场DTC做得最好的外资动保企业，但与国产品牌（海正/瑞普）相比，仍停留在"平台入驻"而非"用户经营"阶段。

DTC战略建议（对勃林格）

  • 建设品牌私域小程序：打通天猫/京东购买数据，建立勃林格自有用户ID，积累数据资产
  • 推出"超可信年框订阅"：基于12周一次的产品特性，推出季度/年度套餐，锁定复购
  • 小红书+抖音内容矩阵：与头部宠物博主建立长期合作，以"科学驱虫=勃林格"抢占Z世代心智
  • 深化医院"导流"机制：医院推荐扫码→品牌小程序复购，形成医院不排斥的DTC闭环
  • 福味恩差异化打法：2025年新品+牛肉味+多宠友好，通过KOL种草快速建立新品认知

与硕腾的根本差异

硕腾的DTC：主动"绕开医院"——因为其产品以驱虫药为主（可线上销售），医院对其而言是"阻力而非助力"

勃林格的DTC："不放弃医院，也不错过线上"——因为其同时拥有妙三多（疫苗，必须依赖医院）等产品，完全绕开医院会顾此失彼

现实评价：在无法复制硕腾美国DTC模式的环境下（妙三多垄断格局+中国医院渠道利益复杂），勃林格的"双轨并行"是最务实的选择，但这意味着DTC的深度（用户数据自持、订阅机制）始终受限。
