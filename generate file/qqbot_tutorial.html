📱 QQ 机器人创建教程

快速获取 App ID 和 App Key · 2026年4月

第一步：注册 / 登录 QQ 开放平台

  1. 访问 QQ 开放平台：https://q.qq.com
  2. 使用 QQ 号登录（如未注册需先完成开发者资质认证）
  3. 建议完成企业开发者认证，功能更全（个人开发者也可创建基础机器人）

第二步：创建应用

  1. 进入「应用管理」→「创建应用」
  2. 选择应用类型：机器人（非小程序/小游戏）
  3. 填写应用基本信息：
    • 应用名称（建议与机器人名一致）
    • 应用描述（不少于20字）
    • 应用图标（建议 256×256 PNG 格式）
  4. 提交审核，等待审批通过
应用名称和头像不要涉及知名品牌、 trademark 或敏感词，否则审核容易被拒。

第三步：获取 App ID 和 App Key

应用创建成功后，在应用详情页即可获取凭证：

凭证名称 字段名 说明
App ID client_id 应用的唯一标识，公开使用
App Key client_secret 应用密钥，请勿泄露给他人

位置示意：应用详情页 → 「开发配置」→「凭证信息」

第四步：获取 Bot Token（API 调用用）

机器人上线后，通过以下接口获取 Access Token：

POST https://api.sgroup.qq.com/oauth2/token 参数（form-data）： grant_type = client_credentials client_id = 你的 App ID client_secret = 你的 App Key

返回示例：

{ "access_token": "xxxxxxxxxxxxxxxx", "expires_in": 7200, "token_type": "Bearer" }

常见问题

问题 解答
AppKey 忘记了怎么办？ 进入应用详情页 → 「开发配置」→ 可重置 AppKey
个人开发者能创建机器人吗？ 可以，但功能受限。企业认证后功能更完整
机器人审核不通过？ 检查应用描述是否完整、头像是否合规、名称是否涉及 trademark
在哪找 Bot Token？ 机器人上线后，通过上方 OAuth2 接口换取，非页面直接展示
