第一步：打开 QQ 开放平台

👉 打开 https://q.qq.com，可选择扫码登录（手机QQ扫）或账号密码登录。新用户点「立即注册」。

第二步：创建应用

👉 登录后点击「立即使用」或「创建应用」，选择「机器人」类型开始创建。

第三步：填写应用信息并提交

按要求填写以下内容后提交审核：

字段说明
应用名称建议与机器人名一致，不要含 trademark
应用描述不少于20字，描述清晰机器人功能
应用图标建议 256×256 PNG 格式
应用名称和头像不要涉及知名品牌、商标或敏感词，否则审核容易被拒。

第四步：获取 App ID 和 App Key

应用审核通过后，在应用详情页 → 开发配置 → 凭证信息获取：

凭证名称 字段名 说明
App ID client_id 应用唯一标识，可公开
App Key client_secret 应用密钥，请勿泄露

第五步：获取 Bot Token（API 调用用）

机器人上线后，通过以下接口换取 Access Token：

POST https://api.sgroup.qq.com/oauth2/token 参数（form-data）： grant_type = client_credentials client_id = 你的 App ID client_secret = 你的 App Key

返回：

{ "access_token": "xxxxxxxxxxxxxxxx", "expires_in": 7200, "token_type": "Bearer" }

常见问题

问题解答
AppKey 忘记了？应用详情页 → 开发配置 → 可重置
个人能创建吗？可以，基础功能可用；企业认证功能更全
审核总被拒？检查名称/头像合规性，描述是否完整清晰
