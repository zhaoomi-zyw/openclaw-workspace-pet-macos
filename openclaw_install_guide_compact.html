环境准备

✅ Homebrew 可用
未安装？
macOS: brew install node
Linux: curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
2
安装 OpenClaw
# 全局安装 npm i -g openclaw # 中国区镜像 npm i -g openclaw --registry=https://registry.npmmirror.com
# 验证 openclaw --version → 2026.3.31
✅ npm global bin 已自动加入 PATH
✅ openclaw help 查看所有命令
3
配置 MiniMax M2.7
# 创建环境配置文件 touch ~/.openclaw/.env
MINIMAX_API_HOST=https://api.minimaxi.com MINIMAX_API_KEY=sk-your-key-here TAVILY_API_KEY=tvly-your-key
# 添加模型（openclaw.json） openclaw models add minimax/MiniMax-M2.7
✅ 当前配置: api.minimaxi.com
✅ 2048k上下文 / 推理模式已开启
4
接入 QQBot 频道
# 安装 QQBot 插件 openclaw plugin install qqbot
# 配置（需 QQ 开放平台 AppID+Token） openclaw qqbot configure # 当前: appId=1903770990
✅ 插件目录: ~/.openclaw/qqbot/
✅ 账号: qqbot-pet-macos
首次配置需在 QQ开放平台 创建机器人，填入 AppID 和 Token
5
安装 Skills
# 安装核心 Skills npx clawhub install web-access npx clawhub install minimax-mcp-call npx clawhub install minimax-multimodal npx clawhub install memory-setup npx clawhub install openclaw-tavily-search
# 查看已安装 openclaw skills list
✅ 当前已安装 6 个 skills
✅ 路径: ~/.openclaw/skills/
